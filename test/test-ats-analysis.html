How to Test the ATS Calculator

  1. Navigate to http://localhost:3003/ats-calculator
  2. Copy one of the sample resume texts above
  3. Paste it into the resume text area
  4. Optionally, copy the job description for better keyword matching
  5. Click "Analyze Resume" to see the results

Expected Results:

Good Resume: Should score 70-85+ overall with high keyword match, good format score, and detailed recommendations.

Poor Resume: Should score 20-40 overall with critical issues flagged and many improvement recommendations.

Features to Test

  • ✅ File upload functionality (.txt files)
  • ✅ Manual text input
  • ✅ Job description input for targeted analysis
  • ✅ Overall ATS compatibility score
  • ✅ Individual scores (Keywords, Format, Content, Readability)
  • ✅ Section scores (Contact, Experience, Education, Skills)
  • ✅ Critical issues, warnings, and suggestions
  • ✅ Missing and found keywords
  • ✅ Prioritized recommendations
  • ✅ Resume metrics (word count, achievements, etc.)
  • ✅ Call-to-action for resume builder
